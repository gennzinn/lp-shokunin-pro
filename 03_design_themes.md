# デザインテーマ集(8種)と選定ロジック

## 選定ロジック

1. 業種で候補を2つに絞る → ターゲットの年齢・性別・価格帯で1つに決める
2. 高単価・信頼重視 → 明朝体系(B/D/F)。低価格・親しみ重視 → ゴシック系(C/E/H)
3. 同じクライアントの2本目以降は**必ず別テーマ**にする(量産感の防止)
4. ユーザーが参考サイトを出したら、そのトーンに一番近いテーマを起点にカスタムする

各テーマの色は `:root` のCSS変数として実装する。
共通実装:ボタンは メイン→アクセントのグラデーション+角丸999px+hoverで浮上。
セクション見出しは「英語ラベル(小・アクセント色・字間.3em)+日本語見出し(大・太字)」の2段。

## A:テックネイビー — IT・AI・DX・SaaS・士業のデジタル系

- 背景 #0b1530(ダーク)⇔ #f5f8ff(ライト)を交互 / メイン #3b82f6 / アクセント #22d3ee
- 文字 #1e2a44・補助 #5a6a8a / 見出し・本文 Noto Sans JP、英字 Montserrat
- ヒーロー:濃紺+薄いグリッド線+光のグラデーション。先進・信頼

## B:トラストグリーン×ゴールド — コンサル・金融・不動産・法律・会計

- 背景 白〜#f3f7f5 / メイン 深緑 #0d5c46 / アクセント 金 #b98f3e
- 文字 #26332e・補助 #5f6f68 / 見出し Noto Serif JP、本文 Noto Sans JP
- 金の細線を区切りに。老舗感・誠実・上品

## C:あたたかオレンジ — 飲食・食品EC・地域産品・観光・農産物

- 背景 クリーム #fffaf3⇔白 / メイン #e8590c / アクセント #f59f00
- 文字 #3d3229・補助 #8a7a6d / 見出し Noto Sans JP 900(丸み希望なら M PLUS Rounded 1c)
- 角丸大きめ・影やわらかめ。親しみ・食欲・地元感

## D:エレガントワイン — 美容・サロン・ブライダル・ジュエリー・高級商材

- 背景 白〜#faf7f8 / メイン ワイン #7d1d3f / アクセント シャンパンゴールド #c9a24b
- 文字 #38262c・補助 #8a6f77 / 見出し Noto Serif JP、英字 Cormorant Garamond
- 余白を贅沢に、装飾は細線と小さな飾り罫のみ。高級・繊細

## E:フレッシュブルー — 医療・歯科・教育・スポーツ・清掃・修理

- 背景 白〜#f2f8fb / メイン #1971c2 / アクセント #15aabf
- 文字 #243a4a・補助 #64808f / 見出し・本文 Noto Sans JP
- 白場を広く、色は差し色程度。清潔・安心・軽快

## F:ミニマルモノクロ — 建築・写真・デザイン事務所・アパレル

- 背景 白 #ffffff⇔ #f4f4f4 / メイン 墨 #1a1a1a / アクセント 1色だけ差す(例 #d43f2a)
- 文字 #1a1a1a・補助 #777 / 見出し Noto Serif JP か Zen Kaku Gothic New、英字 Inter
- 罫線と余白で構成し、装飾ほぼゼロ。写実・洗練。※要素が少ない分、文字組の美しさで勝負

## G:ナイトゴールド — バー・レストラン(高級)・エンタメ・イベント

- 背景 #121212(ダーク基調で通す) / メイン #c9a24b / アクセント #e5c987
- 文字 #f0e9dc・補助 #a89f8d / 見出し Noto Serif JP、英字 Cinzel か Montserrat
- 黒に金文字、写真がなくてもグラデーションの照明感で雰囲気を出す。非日常・特別感

## H:パステルポップ — 子ども向け・教室・ペット・イベント・キッチンカー

- 背景 #fffdf7 / メイン #ff8787 / アクセント #74c0fc(+サブ #ffd43b を少量)
- 文字 #495057・補助 #868e96 / 見出し M PLUS Rounded 1c、本文 Noto Sans JP
- 角丸最大・アイコン多め・チップUI多用。楽しい・やさしい・入りやすい
